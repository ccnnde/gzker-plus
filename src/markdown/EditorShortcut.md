键盘快捷键

| 功能     | 按键(Windows) | 按键(macOS) |
| -------- | ------------- | ----------- |
| 显示帮助 | `Ctrl` + `/`  | `Cmd` + `/` |
| 预览内容 | `Ctrl` + `M`  | `Cmd` + `M` |
| 全屏查看 | `Ctrl` + `F`  | `Cmd` + `F` |
| 历史记录 | `Ctrl` + `H`  | `Cmd` + `H` |
| 提交内容 | `Ctrl` + `⏎`  | `Cmd` + `⏎` |
| 提到用户 | `@`           | `@`         |
| 插入表情 | `Ctrl` + `;`  | `Cmd` + `;` |
| 加粗     | `Ctrl` + `B`  | `Cmd` + `B` |
| 斜体     | `Ctrl` + `I`  | `Cmd` + `I` |
| 删除线   | `Ctrl` + `D`  | `Cmd` + `D` |
| 1 级标题 | `Ctrl` + `1`  | `Cmd` + `1` |
| 2 级标题 | `Ctrl` + `2`  | `Cmd` + `2` |
| 3 级标题 | `Ctrl` + `3`  | `Cmd` + `3` |
| 4 级标题 | `Ctrl` + `4`  | `Cmd` + `4` |
| 5 级标题 | `Ctrl` + `5`  | `Cmd` + `5` |
| 6 级标题 | `Ctrl` + `6`  | `Cmd` + `6` |
| 插入链接 | `Ctrl` + `L`  | `Cmd` + `L` |
| 插入代码 | `Ctrl` + `K`  | `Cmd` + `K` |
| 插入图片 | `Ctrl` + `G`  | `Cmd` + `G` |
